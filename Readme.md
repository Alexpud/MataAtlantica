# Descrição

Projeto para criar o e-commerce "Mata Atlantica" para praticar conceitos de arquitetura de software e nuvem. O projeto está 
sendo desenvolvido em portugues para praticar construir software nessa linguagem e porque o stakeholder(eu) é brasileiro.


## O que vai ter no projeto

A idéia do projeto é desenvolve-lo utilizando uma arquitetura evolutiva, com ela mudando ao decorrer do tempo a medida 
que as necessidades e complexidades do projeto mudam.
